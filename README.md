# Farin TV Display Kiosk Repository

This repository contains the installation script for Farin TV Display units (Raspberry Pi).

## Quick Setup

1. Create a new public repository on GitHub named `tv-kiosk-setup` (or similar).
2. Upload the `install-pi-display.sh` file to that repository.
3. Update the `buildPiScript` function in the main Farin repository (`src/routes/(app)/config/tv-displays/+page.svelte`) to point to your new script:

```javascript
function buildPiScript(code) {
    return `curl -sSL https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/tv-kiosk-setup/main/install-pi-display.sh | bash -s -- --code ${code} --location ${$selectedLocation}`;
}
```

## How to use on a Raspberry Pi

1. Flash **Raspberry Pi OS (64-bit)** to a microSD card.
2. During flashing, use the "Edit Settings" button in Raspberry Pi Imager to:
   - Enable SSH.
   - Configure Wi-Fi.
   - Set a username and password.
3. Boot the Pi and connect via SSH.
4. Run the installation command generated by your Farin dashboard.

## What this script does

- **Registers the device**: Automatically calls the Farin API to claim the setup code.
- **Installs dependencies**: Installs Chromium, unclutter (to hide the mouse), and other tools.
- **Configures Kiosk Mode**: Sets the Pi to automatically open Chromium in full-screen on boot.
- **Installs Remote Agent**: Sets up a background Python service that listens for commands from your dashboard (Reboot, Update OS, etc.).
- **Installs Tailscale**: Prepares the device for secure remote mesh VPN access.

## Remote Management Commands

The background agent supports:
- **Refresh**: Reloads the browser.
- **Clear Cache**: Wipes local storage/cache and reloads.
- **Identify**: Flashes device info on the physical screen.
- **Restart Kiosk**: Kills and restarts the Chromium process.
- **Reboot**: Restarts the entire Raspberry Pi.
- **Update OS**: Runs `sudo apt update && sudo apt upgrade -y` in the background.
